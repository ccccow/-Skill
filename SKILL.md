# 简历格式化处理

## 任务目标
- 本Skill用于：将用户上传的简历文档按照统一格式整理输出
- 能力包含：简历翻译（英文/繁体中文→简体中文）、信息提取与结构化、QS排名查询标注、世界500强标注、格式化表格输出、PDF美化导出
- 触发条件：用户上传简历文档并要求格式化处理、整理简历、生成简历表格

## 输出格式

**简历输出包含两部分**：
1. **Markdown代码块**：包含完整的11个部分表格，便于复制和编辑
2. **美化的HTML/PDF文件**：带样式美化的文档，可直接使用或打印为PDF

---

## 操作步骤

### 步骤1：读取简历文档
- 读取用户上传的简历文件内容
- 识别文档语言（英文/繁体中文/简体中文）

### 步骤2：翻译处理
- 如果是英文或繁体中文，翻译为简体中文
- 保留专业术语、人名、机构名等关键信息的准确性

### 步骤3：查询排名信息
- 对于教育经历中的院校，调用脚本查询QS排名：
  ```bash
  python scripts/query_qs.py "<学校名称>"
  ```
- 对于工作经历中的公司，调用脚本查询世界500强：
  ```bash
  python scripts/query_fortune500.py "<公司名称>"
  ```

### 步骤4：生成Markdown格式简历
按照以下11个部分生成表格，输出为Markdown代码块

### 步骤5：生成HTML/PDF格式简历
- 将Markdown简历保存为临时文件：`./resume.md`
- 调用PDF生成脚本：
  ```bash
  python scripts/generate_pdf.py ./resume.md ./resume.pdf
  ```
- 脚本会生成美化的HTML文件，并提供PDF转换指导
- **PDF样式特点**：
  - 渐变色头部（深蓝色系）
  - 圆角表格边框
  - 高亮编号列
  - 响应式布局
  - 打印优化（A4尺寸）

## 第一部分：基本信息

| 条目 | 内容   | 条目 | 内容  |
|--|--|--|--|
| 姓名 |  | 出生年月 |  |
| 国籍 |  | 所在国家/地区 |  | 
| 研究方向 |  | 博士院校 |  |
| 目前职务|  | 目前单位 |  |

**填写规则**：
- **姓名**：只保留姓+博士二字（如"李博士"、"刘博士"）；外国人用首字母+博士（如"L博士"、"K博士"）；外籍华人保持汉字或拼音
- **出生年月**：优先使用简历提供的信息；若无，按公式计算并标明（估）：出生年份（估）= 本科毕业年份 - 22，或 = 本科入学年份 - 18（以后者为准）
- **国籍**：台湾/香港/澳门写成"中国台湾/中国香港/中国澳门"；中国大陆写成"中国大陆"
- **研究方向**：从简历中总结3个最能代表技术/研究的词汇（1-3个）
- **博士院校**：标注QS排名，格式：斯坦福大学（**26年QS 3**）；QS排名前100加粗；QS-300以外不标注
- **信息缺失**：留空，不写"未提及"等话语

**填写规则**：

|--|--|--|--|
|  |  |  |  |

**填写规则**：
- 工作内容整合为一段话，用分号分隔
- 公司/大学名翻译并保留原名：哈佛大学（Harvard University）、苹果公司（Apple Inc.）

- 时间格式：2013.1 - 2015.9；缺月份则：2013 - 2015

- 标注行业地位：麻省理工学院（**26年QS 1**）、苹果公司（**25年世界500强 第10名**）
- 世界500强查询：使用脚本查询，未查到则不标注

## 注意事项
- 表格行数不够则增加，多余空行则删除
- 只生成简历内容，不写后续建议、反问等
- 信息不全处留空，不写"未提及"、"不详"等话语
- 严格按格式输出，不添加额外内容
- **输出完成后**：
  1. 输出Markdown代码块
  2. 将Markdown保存为文件
  3. 调用脚本生成美化的HTML
  4. 告知用户可打开HTML文件打印为PDF
